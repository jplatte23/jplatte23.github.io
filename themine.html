The Game

My Contributions

  • Tuned and programmed audio manager singleton for all sounds
  • Implemented enemy AI with procedurally generated spider legs
  • Designed all in-game environments and cutscene/story elements
  • Created HLSL shaders for pixel posturize and retro effect
  • Game planned to be released on steam.
